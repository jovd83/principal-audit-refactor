# Security Policy

## Reporting A Vulnerability

If you discover a vulnerability in this repository, please avoid opening a public issue with exploit details.

Instead, report it privately to the repository maintainer with:
- a concise description of the issue,
- affected files or artifacts,
- reproduction steps,
- impact assessment,
- any suggested remediation.

## Scope

This repository contains an Agent Skill and supporting documentation. Security-relevant issues may include:
- unsafe skill instructions that could encourage destructive or insecure behavior,
- exposed secrets or sensitive sample data,
- repository automation that executes untrusted input unsafely,
- examples or templates that normalize insecure patterns.

## Response Expectations

Reports should be acknowledged promptly and triaged based on impact and exploitability.
